# sql-snippets

Handly snippets to create user-defined Tables, Views, Procedures, and Functions to support Data Warehousing and Data Lake Architecture processing.

<h2>Known Issues</h2>
<ul>
  <li>
    <div>
      <h3>SQL Development Environment</h3>

    </div>
  </li>
</ul>
